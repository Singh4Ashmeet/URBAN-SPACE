# UrbanShield ML Model Card

No ML service is implemented yet in the current Phase 4 foundation.

## Planned Purpose

The planned ML service will provide experimental local predictions for:

- incident risk score
- estimated response time
- recommended vehicle count

## Planned Data

Training data must be synthetic, deterministic, and clearly labeled as non-real-world.

## Limitations

- No model should be presented as operationally accurate.
- Outputs must be advisory only.
- Operators must be able to override recommendations.

## Status

Not implemented.
